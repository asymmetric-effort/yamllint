@asymmetric-effort/yamllint

A pure TypeScript YAML linter with configurable rules and detailed error reporting.

npm install -g @asymmetric-effort/yamllint

24 Rules

Comprehensive checks covering formatting, style, and correctness for all YAML patterns.

Configurable

Built-in default and relaxed presets. Override any rule with severity levels.

Multiple Formats

Standard, parsable, colored, and GitHub Actions output formats.

Comment Directives

Suppress rules inline with disable-line, block disable/enable, and disable-file.

TypeScript

Full programmatic API with complete type definitions for integration.

CI Ready

Strict mode, parsable output, and GitHub Actions annotations built in.

Usage

# Lint files
yamllint myfile.yaml
yamllint ./config/

# Custom configuration
yamllint -c .yamllint.yaml .
yamllint -d "extends: relaxed" .

# CI-friendly output
yamllint -f parsable --strict .

# Read from stdin
cat file.yaml | yamllint -

Configuration

# .yamllint.yaml
extends: default

rules:
  line-length:
    max: 120
    level: warning
  document-start: disable
  truthy:
    allowed-values: ["true", "false"]
